### virtualenv

1. **install**
    ```bash
    sudo pip3 install virtualenv    
    ```

2. **project directory create**
    ```bash
    mkdir myproject
    cd myproject
    ```

3. **env_create**
    ```bash
    virtualenv <env_name>
    ```

4. **activate env**
    ```bash
    source <env_name>/bin/activate
    ```

5. **deactivate env**
    ```bash
    deactivate
    ```
